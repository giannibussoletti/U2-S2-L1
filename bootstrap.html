Bootstrap

introduzione a bootstrap

È un framework che permette di creare velocemente dei siti responsive dando codice già pronto da poter utilizzare.

framework vs libreria

  • libreria: aiuta a risolvere piccoli problemi specifici come per esempio animazioni o date
  • framework: lo sviluppatore si immerge completamente nel framework e si trova a lavorare con gli strumenti che il framework fornisce. Nei framework è quest'ultimo che comanda

Cosa è Bootstrap

è un foglio CSS pronto che si applica di base al nostro documento. È considerato uno dei migliori framework front-end tutt'oggi

Non c'è bisogno di riscrivere media queries per esempio perché sono già inserite nel framework.

Cosa c'è dentro Bootstrap?

Come già detto BS ha pezzi di codice già pronto che devono essere copiati ed incollati. Ma Il "problema" di bootstrap è che non è molto "personalizzabile" a livello di impaginazione.
